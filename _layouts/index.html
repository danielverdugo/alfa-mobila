Тестовое задание
дизайнера интерфейсов
для «Альфа-Лаборатории»

Задача

Создать прототип интерфейса оплаты счета мобильного телефона для Альфа-Банка и показать пользовательское взаимодействие. Для простоты и скорости ограничиться мобильной веб-версией.
Письмо
с заданием

Прототип

Для решения задачи сделан HTML-прототип. В отличие от статичных картинок, прототип наглядно показывает поведение элементов  при взаимодействии (анимации, переходы, подсветка ошибок и т.д.), позволяет тестировать интерфейс «в поле» и облегчает работу разработчиков фронт-энда.
Открыть прототип в браузере


Платежи для всех

 Банку выгодно проводить платежи, в том числе с карт других банков — это приносит комиссионый доход. Для гостя в форму добавятся поля ввода данных карты и информация о комиссии.




Автоопределение оператора

Оператор определяется по первым 5 цифрам номера. Если определение не сработало или сработало неверно, оператор выбирается из меню.


Подсветка ошибок

Неподходящие значения сразу подсвечиваются при вводе.


Модульный дизайн

Оформление элементов подобрано с учетом нового оформления сайтов «Альфа-Банка». Элементы расставлены по заданной модульной сетке, легко воспроизводимой при вёрстке.


С вниманием к типографике

Числа правильно разбиваются на разряды, используются шпации правильной ширины и специальные типографские знаки. Размер букв подобран в соответствии с сеткой, межбуквенные расстояния откернены.

Программы и технологии


      

HTML, CSS (SASS), JavaScript, jQuery, Git
